# Backend - Book Reader API

Node.js API server for book processing and AI integration.

## Technology Stack
- Node.js + Express
- TypeScript
- PostgreSQL (later) / JSON files (initial)
- Redis (caching)

## Features to Implement

## Setup
```bash
cd backend
npm install
npm run dev
```
