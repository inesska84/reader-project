# Frontend - Book Reader UI

React application for the book reader interface.

## Technology Stack
- React 18 + TypeScript
- Vite (build tool)
- Tailwind CSS (styling)
- Zustand (state management)

## Features to Implement

## Setup
```bash
cd frontend
npm install
npm run dev
```
